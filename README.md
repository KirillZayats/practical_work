## 2020.06.16 - 2020.06.17
- настроить [webpack](https://webpack.js.org/guides/getting-started/) - ВАЖНО
- подключить scss вместо css
- все шрифты должны лежать в папке fonts
- все картинки в папке images
- svg можно хранить в папке svg

- в папке src/ должен находится весь исходный код. рекомендуется хранить scss файлы в папке src/styles/ константы в папке src/contants, компоненты в папке rc/components и так далее
